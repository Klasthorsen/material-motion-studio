# Material Motion Studio

Use this skill together with the local Motion Studio app in `studio/`. It is best for collaborative sessions where a component or screen is reviewed, a preset is adjusted, and the approved motion is then documented back into the motion library.

## Workflow

1. Pick a pattern in the studio.
2. Adjust timing, easing, movement, opacity, and scale.
3. Decide whether the result is a `standard` or `expressive` motion behavior.
4. Document the approved values in the shared guideline.

## Focus Areas

- Dashboard transitions
- KPI drill-downs
- Filter changes
- Tab switches
- Overlays and sheets
